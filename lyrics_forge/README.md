## 🚀 Usage & Workflows

### Standard Pipeline
1. **Conceptualize**: Generate raw thematic concepts using `llm_forge`.
2. **Refine**: Structure the output into verses/choruses within `lyrics_forge`.
3. **Execute**: (Future) Pipe the finished structure to external audio generation APIs (e.g., Suno/Suno_forge).
