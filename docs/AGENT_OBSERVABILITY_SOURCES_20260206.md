# Agent Observability Sources (2026-02-06)

This note collects authoritative sources used for agent observability and auditability guidance.

## Logging and Audit Controls

- NIST SP 800-92: Guide to Computer Security Log Management (final, 2006)
  - https://doi.org/10.6028/NIST.SP.800-92
  - Publication page: https://csrc.nist.gov/pubs/sp/800/92/final
- NIST SP 800-92 Rev. 1 (Initial Public Draft): Cybersecurity Log Management Planning Guide (2023)
  - https://doi.org/10.6028/NIST.SP.800-92r1.ipd
  - Publication page: https://csrc.nist.gov/pubs/sp/800/92/r1/ipd
- NIST Log Management Project overview (context and scope)
  - https://csrc.nist.gov/projects/log-management
- NIST SP 800-53 Rev. 5 controls for audit logging
  - AU-2 Event Logging: https://nist-sp-800-53-r5.bsafes.com/docs/3-3-audit-and-accountability/au-2-event-logging/
  - AU-4 Audit Log Storage Capacity: https://nist-sp-800-53-r5.bsafes.com/docs/3-3-audit-and-accountability/au-4-audit-log-storage-capacity/
  - AU-12 Audit Record Generation: https://nist-sp-800-53-r5.bsafes.com/docs/3-3-audit-and-accountability/au-12-audit-record-generation/
